# 💰 Mr. Finance – Smart Expense Tracking App

--- 
**Mr. Finance** is a Flutter-based financial management application that scans **receipt images** and converts them into **meaningful financial insights**.
The app helps users **track, analyze, and manage expenses efficiently** by generating automated summaries, analytics, alerts, and visual reports.

The system leverages **OCR and data extraction techniques** to read receipt data and categorize spending.

---

## 🚀 Key Features

* 📸 **Receipt Scanning**

  * Scan and upload receipt images.
  * Extract key financial details (date, amount, merchant).
* 📊 **Monthly Expense Summaries**

  * Automatically generate monthly spending summaries.
* 📈 **Spending Analytics**

  * Analyze expenses by category and time period.
* 🔔 **Smart Alerts**

  * Notify users when spending exceeds predefined limits.
* 📉 **Visual Reports**

  * Interactive charts and graphs for better financial insights.

---


## 📱 App Workflow

1. User scans or uploads a receipt image.
2. OCR extracts text from the receipt.
3. Expense data is processed and categorized.
4. Monthly summaries and analytics are generated.
5. Visual reports and alerts are displayed to the user.

---


## 🎯 Future Enhancements

* AI-based receipt categorization
* Cloud backup and multi-device sync
* Export reports as PDF or Excel
* Budget planning and savings goals
* Multi-currency support

# 🌍 Air Detector – Flutter App

A **one‑page Flutter application** that detects and displays **air pollution levels** based on the user’s current location using a public Air Quality API.

---

## 🚀 App Overview

**Air Detector** allows users to instantly check air pollution data for their location. The app automatically fetches the user’s location, calls an Air Quality API, and displays pollution levels in a clean, readable table format.

This project is ideal for:

* Flutter beginners & intermediates
* It's a University hackathon submissions

---

## 📱 Key Features

* 📍 Auto-detect user location (GPS)
* 🌫 Fetch real-time air pollution data via API
* 📊 Display pollution levels in table format
* ⚡ One-page, fast & lightweight UI
* 🎥 App demo video included

---

## 🧪 Pollution Data Displayed

The app shows air quality results based on API response, such as:

| Parameter | Description               |
| --------- | ------------------------- |
| AQI       | Air Quality Index         |
| PM2.5     | Fine particulate matter   |
| PM10      | Coarse particulate matter |
| CO        | Carbon Monoxide           |
| NO2       | Nitrogen Dioxide          |
| O3        | Ozone                     |
| SO2       | Sulfur Dioxide            |

> Values and categories (Good, Moderate, Poor, etc.) are derived directly from the API response.
